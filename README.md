# PullPod 🫛

**A pod for every pull request.** Internal Slack ↔ GitHub app for the Voovo team, inspired by [Axolo](https://axolo.co): every PR gets its own ephemeral Slack channel that syncs comments, reviews, and CI status — then archives itself on merge.

📋 See [plan.md](plan.md) for the full technical plan (architecture, data model, event flows, phased roadmap).
